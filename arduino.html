Arduino

Belajar pemrograman mikrokontroler dan elektronika dasar

Apa itu Arduino?

Arduino adalah platform pemrograman mikrokontroler open-source yang dirancang khusus untuk membuat proyek elektronik interaktif. Program ini cocok untuk anak usia 10+ tahun yang ingin belajar coding sambil membuat perangkat elektronik nyata.

Siswa akan belajar bahasa pemrograman C/C++ melalui Arduino IDE, memahami konsep sensor dan aktuator, serta membuat berbagai proyek seperti lampu LED interaktif, sistem alarm, robot sederhana, dan banyak lagi. Arduino menjembatani dunia digital (software) dengan dunia fisik (hardware).

Tools yang Dipakai

  • Arduino Board (Uno, Nano, Mega)
  • Arduino IDE (Software Pemrograman)
  • Breadboard & Kabel Jumper
  • LED, Resistor, dan Komponen Elektronik
  • Sensor (Ultrasonik, LDR, PIR, dll)
  • Motor DC, Servo, dan Buzzer

Materi yang Dipelajari

Dasar Pemrograman C/C++

Mempelajari syntax dasar, variabel, tipe data, operator, kondisi (if-else), dan perulangan (loop) dalam bahasa C/C++.

Digital I/O & Analog

Memahami pin digital dan analog pada Arduino, cara mengontrol LED, membaca tombol, dan menggunakan PWM.

Sensor & Input

Belajar menggunakan berbagai sensor seperti sensor jarak (ultrasonik), sensor cahaya (LDR), dan sensor gerak (PIR).

Aktuator & Output

Mengontrol motor DC, servo motor, buzzer, dan LCD display untuk membuat output yang interaktif.

Serial Communication

Memahami komunikasi serial antara Arduino dan komputer untuk debugging dan monitoring data real-time.

Proyek IoT & Robotik

Membuat proyek nyata seperti sistem smart home, robot obstacle avoidance, weather station, dan banyak lagi.

Program yang Digunakan

Software dan platform yang digunakan dalam pembelajaran Arduino

Arduino IDE

Software utama untuk menulis, compile, dan upload code ke board Arduino

C/C++ Language

Bahasa pemrograman yang digunakan untuk menulis logic dan algoritma

Serial Monitor

Tool untuk debugging dan monitoring data real-time dari Arduino

Library Manager

Akses ke ribuan library untuk mempermudah penggunaan sensor dan modul

Fritzing

Software untuk desain skematik rangkaian elektronik dan PCB layout

Arduino Cloud

Platform IoT untuk membuat proyek Arduino yang terhubung ke internet
