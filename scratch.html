Scratch

Belajar pemrograman visual yang menyenangkan untuk anak-anak

Apa itu Scratch?

Scratch adalah bahasa pemrograman visual yang dikembangkan oleh MIT (Massachusetts Institute of Technology) khusus untuk anak-anak usia 8 tahun ke atas. Dengan Scratch, anak-anak belajar coding dengan cara menyusun blok-blok perintah seperti menyusun puzzle.

Melalui Scratch, siswa dapat membuat game interaktif, animasi, cerita digital, simulasi, dan berbagai proyek kreatif lainnya. Scratch mengajarkan konsep pemrograman fundamental seperti sequence, loop, kondisi, dan event dengan cara yang visual dan menyenangkan.

Tools yang Dipakai

  • Scratch Desktop (Aplikasi Offline)
  • Scratch Online (scratch.mit.edu)
  • Block Programming Interface
  • Sprite Editor & Paint Tool
  • Sound Editor & Music Library
  • Backdrop & Asset Library

Materi yang Dipelajari

Block Programming

Mengenal berbagai jenis blok perintah: Motion, Looks, Sound, Events, Control, Sensing, Operators, dan Variables.

Loop & Repetition

Memahami konsep pengulangan (repeat, forever, repeat until) untuk membuat kode lebih efisien.

Conditional & Logic

Belajar if-else statement, logical operators, dan decision making dalam programming.

Game Development

Membuat berbagai jenis game seperti platformer, maze, quiz, catching game, dan shooting game.

Animation & Story

Membuat animasi karakter, cerita interaktif, dan presentasi multimedia dengan Scratch.

Variables & Broadcasting

Menggunakan variabel untuk menyimpan data dan broadcasting untuk komunikasi antar sprite.

Program yang Digunakan

Platform dan tools yang digunakan dalam pembelajaran Scratch

Scratch Desktop

Aplikasi offline untuk membuat proyek Scratch tanpa koneksi internet

Scratch Online

Platform web-based di scratch.mit.edu untuk coding dan sharing proyek

Block Editor

Interface drag-and-drop untuk menyusun blok kode dengan mudah

Paint Editor

Tool untuk menggambar dan mengedit sprite, backdrop, dan costume

Sound Editor

Tool untuk mengedit audio, menambahkan sound effects dan music

Asset Library

Koleksi sprite, backdrop, sound, dan costume siap pakai
