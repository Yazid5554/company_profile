WeDo

Belajar Robotik Interaktif dengan LEGO WeDo 1.0 & 2.0

Apa itu WeDo?

LEGO WeDo adalah program pembelajaran robotik untuk anak-anak usia 7-10 tahun yang menggunakan kit konstruksi LEGO khusus dengan motor dan sensor. Program ini dirancang untuk memperkenalkan konsep dasar robotika, mekanika, dan pemrograman visual dengan cara yang menyenangkan dan interaktif.

DC Robotic menggunakan WeDo 1.0 dan WeDo 2.0 untuk memberikan pengalaman pembelajaran yang komprehensif. Anak-anak belajar membangun robot sederhana menggunakan balok LEGO, memasang motor dan sensor, kemudian memprogram robot mereka untuk melakukan berbagai aksi seperti bergerak, berputar, mendeteksi gerakan, dan mengeluarkan suara.

Perbedaan WeDo 1.0 & 2.0

WeDo 1.0

  • Koneksi USB ke komputer
  • 1 Motor dan 2 Sensor (Motion & Tilt)
  • 158 balok LEGO
  • Software berbasis Scratch

WeDo 2.0

  • Koneksi Bluetooth/wireless
  • SmartHub terintegrasi
  • 280+ balok LEGO
  • Aplikasi lebih modern & user-friendly

Komponen LEGO WeDo

LEGO Bricks

158-280+ balok LEGO berwarna untuk membangun robot

USB Hub / SmartHub

Otak robot yang menghubungkan motor dan sensor ke komputer

Motor & Gears

Motor listrik dan sistem gear untuk menggerakkan robot

Motion Sensor

Mendeteksi objek dan gerakan di sekitar robot

Tilt Sensor

Mendeteksi kemiringan dan orientasi robot

WeDo Software

Platform pemrograman visual drag-and-drop

Materi yang Dipelajari

Konstruksi Robot

Membangun struktur robot dengan balok LEGO, memasang motor, sensor, dan memahami mekanisme roda serta gear untuk menciptakan robot yang stabil dan fungsional.

Pemrograman Visual

Belajar coding dengan blok visual drag-and-drop, logika sekuensial, penggunaan loop (pengulangan), dan kondisi sederhana (if-then) untuk mengontrol robot.

Sensor & Interaksi

Menggunakan motion sensor dan tilt sensor untuk membuat robot yang dapat merespons lingkungan sekitar dengan suara, lampu, dan gerakan.

Konsep STEM

Mengintegrasikan Science (gaya & gerak), Technology (motor & sensor), Engineering (desain robot), dan Math (pengukuran & sudut) dalam satu proyek.

Project-Based Learning

Membuat proyek nyata seperti robot hewan, mesin sederhana, simulasi bencana alam, lengkap dengan presentasi dan storytelling.

Program yang Digunakan

LEGO WeDo Software

  • Interface visual drag-and-drop yang mudah dipahami
  • Library proyek dengan panduan step-by-step
  • Coding blocks untuk motor, sensor, suara, dan display
  • Documentation tool untuk membuat laporan proyek
  • Compatible dengan Windows, macOS, iOS, Android, Chromebook
  • Mendukung WeDo 1.0 dan WeDo 2.0

Scratch (Optional)

  • Platform pemrograman visual dari MIT
  • Dapat diintegrasikan dengan LEGO WeDo
  • Lebih fleksibel untuk proyek kreatif
  • Berbasis komunitas global
  • Cocok untuk yang sudah familiar dengan WeDo
